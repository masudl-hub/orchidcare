## üé® Visual Language

### Color Palette
- **Primary**: Stone/warm grays (`#78716c`, `#57534e`, `#44403c`)
- **Background**: Cream/off-white (`#fafaf9`, `#f5f5f4`)
- **Accents**: Pure black (`#000000`) and white (`#ffffff`)
- **Nature tones**: Emerald greens in video backgrounds only

### Typography Style
- **Headlines**: Large serif fonts (like Playfair Display or similar editorial serif)
- **Body/UI**: Monospace uppercase with wide letter-spacing
- **Labels**: 10px monospace, uppercase, tracking-widest

### Design Elements
- **Hard shadows**: `offset-x: 4-8px, offset-y: 4-8px, color: black`
- **Borders**: 2px solid black on all UI elements
- **Rounded corners**: Subtle (8px radius)
- **Etched botanical illustrations**: Fern, Monstera, Palm, Leaf as subtle background textures
